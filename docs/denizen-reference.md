## Denizen Script References

Denizen script is a scripting language that is supported on Spigot/Bukkit/PaperMC servers through plugins. There is a LOT of information about this on the internet, and the language is highly flexible. Some quick reference links are available below.

- [Denizen Scripts Link Directory](https://denizenscript.com/)
- [Denizen Beginner Guide](https://guide.denizenscript.com/)
- [Denizen Language Reference](https://meta.denizenscript.com/)
- [External plugin integrations](https://github.com/DenizenScript/Depenizen)
- [Denizen Discord bot support](https://github.com/DenizenScript/dDiscordBot)
